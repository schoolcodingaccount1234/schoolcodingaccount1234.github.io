Projects

In 2022, I decided to independently study Java using a tutorial. I learned how to make a snake game that opened in it's own window. The green snake grows as it eats more red apples, and dies if it crashes into anything else, including itself.

I decided that the tutorial was cool, but it could use more features, so I added difficulty controls and a display of your previous score.

Even on the highest difficulty, the game was still fairly predictable and easy if you played it a few times, so I decided to add orange enemies that chase you.
